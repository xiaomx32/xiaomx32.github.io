概率论的基本概念

发表于2021-04-13|更新于2023-04-05
|阅读量:

事件的包含关系与事件的运算

  1. 事件的包含与相等
    如果事件$A$发生必然导致事件$B$发生，则称事件$B$包含事件$A$.记为$B \supset A$或$A \subset B$.

    若$A \subset B$且同时$B \supset A$，则称事件$A$与事件$B$相等(等价)，记为$A = B$.

  2. 事件的和(并)
    事件$A \cup B = { x| x \in A 或 x \in B }$称为事件$A$与事件$B$的和事件(或并事件).当且仅当事件$A$与事件$B$至少一个发生时，事件$A \cup B$发生.
    推广 $A_1 \cup A_2 \cup \dots \cup A_n = \bigcup\limits_{i=1}^{n} A_i$称为事件$A_1, A_2, \dots, A_n$的和事件.

  3. 事件的积(交)
    事件$A \cap B = AB = { x| x \in A 且 x \in B}$称为事件$A$与事件$B$的积事件(或交事件).当且仅当事件$A$与事件$B$同时发生时，事件$A \cap B$发生.
    推广 $A_1 \cap A_2 \cap \dots \cap A_n = \bigcap\limits_{i=1}^{n} A_i$称为事件$A_1, A_2, \dots, A_n$的积事件.

  4. 两事件的差事件
    事件$A - B = {x| x \in A 且 x \notin B}$称为事件$A$与事件$B$的差事件.当且仅当事件$A$发生，事件$B$不发生时，事件$A - B$发生.

  5. 互不相容事件(互斥事件)
    若$A \cap B = \varnothing$(即$A, B$两事件不可能同时发生)，则称事件$A, B$为互不相容事件.

  6. 互逆事件(相互对立事件)
    若$A \cap B = \varnothing 且A \cap B = S$，则称事件$A$与事件$B$互为逆事件或互为对立事件.
    注：$A$与$\overline{A}$互为对立事件\leftrightarrow$A \cap \overline{A} = \varnothing$,$A \cup \overline{A} = S$.此外$\overline{\overline{A}} = A$
